The grid system in Schmutz is called blocks and is on a scale of 1-12

The parent div needs a class of blocks (or blocks-x) reverse-blocks-x, blocks-y or reverse-blocks-y

Blocks

Parent div has class "blocks-x" or just "blocks" for a horizontal set of blocks. Each block in this row has the same defined height and width. Extra space is divided up between the blocks (like margin)

All children have the same definied height and width. Parent div has class blocks-break. Notice only blocks that can't fit on a line move to the next and have as much space between them as possible.

All children have the same definied height and width. Parent div has class reverse-x-blocks.

There are quite a bit more traditional classes available. If you are familiar with flexbox, these may be helpful:

.center-blocks { justify-content: center; }
.start-blocks { justify-content: start; }
.end-blocks { justify-content: end; }
.space-between-blocks { justify-content: space-between; }
.space-evenly-blocks { justify-content: space-evenly; }
.space-around-blocks { justify-content: space-around; }
.center-blocks-secondary { align-items: center; }
.stretch-blocks-secondary { align-items: space-between; }
.start-blocks-secondary { align-items: space-evenly; }
.end-blocks-secondary { align-items: space-around; }
.base-blocks-secondary { align-items: baseline; }
.blocks-content-start { align-content: flex-start; }
.blocks-content-end { align-content: flex-end; }
.blocks-content-center { align-content: center; }
.blocks-content-stretch { align-content: stretch; }
.blocks-content-space-between { align-content: space-between; }
.blocks-content-space-around { align-content: space-around; }
.wrap-blocks { flex-wrap: wrap; }
